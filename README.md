# Codeigniter-Controller-Snippet

Bu kod, personel yönetim sisteminde bir personelin şirket dışında bir görevi olduğu zaman doldurduğu formdur. 
-Bu dış görev ve araç kullanma talebi, birim amiri taranfından onaylandıktan sonra, personel şirket araçlarından o an müsait olanı alıp belirlediği saatler arasında dış görevini yapabilir. Personel dış görevden geri döndüğü zaman, "geri döndüm" butonuna basması gerekmektedir. Yazılan geri dönüş tarihi tahminidir, gerçek hayatta planlara uyulmayabilir. Personelin bastığı "geri döndüm" butonuna ne zaman basıldığı dış görevin süresini hesaplamada baz alınır. 

Ekran görüntüsü olarak bu Controller'ın bağlı olduğu View da verilmiştir.
